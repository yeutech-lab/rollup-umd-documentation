### BREAKING CHANGES

* **theme:** Without correct theme variable component will not be able to be rendered correctly.
* **theme-styleguide:** Whitout the theme.js components will make errors such as variable is not defined
* **theme:** Without coorect variable, rsg-components will throw errors
* **package.json:** If not installed it will return an error as babel-core is missing
* **collapse-menu:** It will create unworked ui component cause of missing variables from components and
theme
* **package.json:** babel-core will be required for build
* **package-lock.json:** Without revised package-lock.json npm install cmd will throw an error
* **components:** If image is not present it will throw an error that image cannot be imported
* **rsg-components:** If loader is not present it will throw an error that imported file do not exist
* **rsg-components:** If theme don't have rsg-components variables it will throw an error that style
variable is unefined
* **css-loader, file-loader:** If dependencies are not isntalled in dev dependencies app will return error not
found file
* **component:** If defaultLogo is not used an error will appear cause it will try to import new
logo not commited yet
* **test improve:** Test will return an error without this update
